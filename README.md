# QuickMark - 极简收藏启动器 🚀

一款基于 Chrome Manifest V3 的精美书签管理扩展，帮你快速访问和搜索浏览器收藏夹。

## ✨ 功能特性

- **📂 书签浏览** — 自动读取 Chrome 书签，以卡片网格形式清晰展示
- **🔍 实时搜索** — 输入关键词即可按标题或网址快速筛选书签（120ms 防抖）
- **🏷️ 智能分类** — 搜索框下方自动按网站类型分组（社交、视频、开发、购物、资讯、AI、工具），一键筛选
- **📊 使用统计** — 自动记录点击次数，按使用频率排序，常用书签靠前展示
- **🎨 多列布局** — 支持 2/3/4 列网格切换，适应不同屏幕宽度
- **🖼️ 网站图标** — 自动加载每个书签的 Favicon，加载失败时显示首字母默认图标
- **⚡ 轻量流畅** — 纯原生 JavaScript + CSS，无外部依赖，启动即用

## 🛠️ 技术栈

| 项目 | 说明 |
|------|------|
| 清单版本 | Manifest V3 |
| 权限 | `bookmarks`、`storage`、`tabs` |
| 存储 | `chrome.storage.local`（持久化使用次数和列数设置） |
| 图标 | Google Favicons API |

## 📦 安装

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启 **开发者模式**（右上角开关）
3. 点击 **"加载已解压的扩展程序"**，选择本项目所在文件夹

## 🖱️ 使用方式

- 点击扩展图标打开弹出面板
- 点击任意书签卡片即可在新标签页中打开
- 顶部的下拉框可切换卡片列数（2/3/4 列）
- 搜索框支持实时过滤，匹配书签标题或网址

## 📄 文件结构

```
quickmark-extension/

├── icon.png        # 扩展图标
└── README.md       # 本文档
```
